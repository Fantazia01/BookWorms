# BookWorms
## Setup
1. Clone the repository
2. Navigate to the backend folder
3. Run these steps:

```sh
pip install -r requirements.txt
```
```sh
python create_db.py
```
```sh
uvicorn main:app --reload
```
4. Open a new terminal and navigate to the frontend repository (Don't close the previous one)
5. Run these steps:
```sh
npm install
```
```sh
npm run dev 
```
